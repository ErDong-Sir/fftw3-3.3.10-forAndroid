# fftw3-3.3.10-forAndroid
Use the latest version of fftw to compile the Android static library warehouse

# how to use?
## step1
    git clone xxxxxxxxxxxxxxxxxx.git

## step2
    run build.sh ,ndk will be automatically downloaded and configured

## step3
    The static library will be generated in the out_libs/ directory

# hint
    Single-precision, neon-accelerated static library used by default
